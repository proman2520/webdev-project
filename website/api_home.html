The API can be accessed through a number of routes to return JSON data. The data is stored in a SQL database and is then pulled by Flask to be interpreted. Some of these routes include:


API Route Returns
127.0.0.1:5000/api/v1.0/all_data Returns a JSON dictionary of all sightings.
127.0.0.1:5000/api/v1.0/all_locations Returns a JSON dictionary of all coordinates.
127.0.0.1:5000/api/v1.0/id/[id] Returns the entry associated with a specific #ID.
127.0.0.1:5000/api/v1.0/state/[state abbreviation] Returns the number of sightings and average sighting duration for a given state.
127.0.0.1:5000/api/v1.0/city/[city]/[state abbreviation] Returns the number of sightings and average sighting duration for a given city.
127.0.0.1:5000/api/v1.0/year/[year] Returns the number of sightings and average sighting duration for a given year.
127.0.0.1:5000/api/v1.0/month-year/[month]/[year] Returns a JSON dictionary of all sightings from a specific month of a given year.
127.0.0.1:5000/api/v1.0/month-year/duration/[duration] Returns a JSON dictionary of all sightings below a given duration in seconds.
127.0.0.1:5000/api/v1.0/month-year/shape/[descriptor] Returns a JSON dictionary of all sightings associated with the same descriptor.
